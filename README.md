# まいばすラリー（東京版） 🛒

東京都内の「まいばすけっと」を巡る、位置情報スタンプラリーアプリです。
在宅勤務の運動不足解消のために、生成AI（Gemini）とペアプログラミングで開発しました。

**👉 [アプリを開く](https://[あなたのGitHubユーザー名].github.io/[リポジトリ名]/)**
*(※公開後にここにURLを貼ってください)*

## 📱 特徴
- **800店舗以上を網羅**: 東京都内の店舗データを収録（2025年11月時点）。
- **消印風スタンプ**: チェックインすると、店名・日付・時間が刻印されたスタンプが押されます。
- **PWA対応**: ホーム画面に追加することで、ネイティブアプリのように全画面で動作します。
- **プライバシー重視**: 位置情報やスタンプ履歴は端末内（ブラウザ）にのみ保存され、外部サーバーには送信されません。

## 🛠️ 技術スタック
- HTML5 / CSS3 / JavaScript (Vanilla JS)
- [Leaflet.js](https://leafletjs.com/) (地図表示)
- [OpenStreetMap](https://www.openstreetmap.org/) (地図データ)
- JSON (店舗データ管理)
- **Backend-free**: サーバーサイド処理を含まない静的サイト構成です。
